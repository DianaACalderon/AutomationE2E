Configuración del Proyecto:

Framework: Serenity BDD
Build Tool: Gradle
IDE: IntelliJ IDEA
Java Version: 1.8, 11, o 17

Archivos del Proyecto:

src/test/java para los tests en Serenity.
src/test/resources para los archivos de configuración y datos de prueba.
